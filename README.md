# Chat_AI 🤖💬

**Chat_AI** is a modern web-based chat application designed to integrate **AI-powered conversational intelligence**.
The project focuses on building a clean, responsive UI with scalable architecture, making it easy to extend with AI models, APIs, and real-time features.

---

## ✨ Features

* 💬 Interactive chat interface
* 🤖 AI-ready architecture (easy to connect with LLM APIs)
* ⚡ Fast and responsive UI
* 🎨 Styled using **Tailwind CSS**
* 🧩 Component-based structure (React)
* 🛠 Clean and extensible codebase
* 📦 Managed with **npm**
* 🔒 MIT Licensed (open for reuse)

---

## 🧱 Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Package Manager:** npm
* **Build Tool:** Create React App
* **Version Control:** Git & GitHub

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/mh-fahim-070/Chat_AI.git
cd Chat_AI
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Start the development server

```bash
npm start
```

The app will run at:

```
http://localhost:3000
```
